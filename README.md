# Xuss

Xuss is a test instrument (engine-speed edge trains, input forcing, current measurement) for Python-based embeeded device development. 

The name is the short half of Turminder Xuss, the drone in Iain M. Banks' *Matter*: precise dirty work while the human keeps judgment.

## Status

**Spec only.** [spec.md](spec.md) is the contract. The firmware gets built from it by an agent, on a branch, with an ambiguity log using https://github.com/tig/silico.

Hardware is an M5Stack M5GO IoT Starter Kit v2.7.
